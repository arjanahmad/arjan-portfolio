## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/arjanahmad/arjan-portfolio.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build for production

```bash
npm run build
```
